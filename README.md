# NovaAI – Responsive Landing Page

NovaAI is a modern, responsive landing page for a fictional AI-powered SaaS platform.  
It was built using **HTML5** and **CSS3**, focusing on clean UI design, responsiveness, and modern web aesthetics like glassmorphism and gradient-based styling.

---

## 🚀 Features

- Fully responsive layout (desktop, tablet, mobile)
- Modern glassmorphism navigation bar
- Hero section with call-to-action buttons
- Feature grid showcasing services
- Pricing section with 3-tier plans
- Testimonials section
- FAQ section
- Contact form UI
- Smooth modern UI styling with gradients and blur effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox + Grid)
- Google Fonts (Inter)

---

## 🎯 Project Purpose

This project was created as a **front-end practice project** to demonstrate:

- Layout structuring with HTML
- Responsive design using CSS Grid and Flexbox
- Modern UI styling techniques
- Building real-world SaaS-style landing pages

---

## 📱 Responsiveness

The website is optimized for:
- Desktop screens
- Tablets
- Mobile devices
